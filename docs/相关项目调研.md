## 二、Clawra 原版详细分析

### 2.1 项目概述

**Clawra** 是 SumeLabs 开发的 OpenClaw 自拍技能，让 AI Agent 能够：
- 生成一致外观的自拍图片
- 跨所有消息平台发送图片（Discord, Telegram, WhatsApp 等）
- 视觉化回应"你在做什么？"、"发张照片"等请求

### 2.2 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│                      Clawra 架构                             │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐     ┌─────────────┐     ┌─────────────┐   │
│  │  fal.ai API │────▶│ xAI Grok    │────▶│  图片生成   │   │
│  │  (接口层)   │     │  Imagine    │     │             │   │
│  └─────────────┘     └─────────────┘     └─────────────┘   │
│                                                 │           │
│                                                 ▼           │
│  ┌─────────────┐     ┌─────────────┐     ┌─────────────┐   │
│  │ OpenClaw    │────▶│ Gateway API │────▶│ 多平台发送  │   │
│  │ Agent       │     │             │     │             │   │
│  └─────────────┘     └─────────────┘     └─────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 自拍模式

| 模式 | 适用场景 | 触发关键词 |
|------|---------|-----------|
| **Mirror（镜面）** | 全身照、穿搭展示 | wearing, outfit, fashion |
| **Direct（直拍）** | 近景、地点背景 | cafe, beach, portrait, smile |

### 2.4 核心特性

| 特性 | 说明 |
|------|------|
| **参考图一致性** | 使用固定 CDN 参考图确保外观一致 |
| **多平台支持** | Discord, Telegram, WhatsApp, Slack, Signal, MS Teams |
| **一键安装** | `npx @sumelabs/clawra` 自动配置 |
| **SOUL 注入** | 自动添加自拍人设到 Agent |

### 2.5 使用示例

```
"Send me a selfie"
"Send a pic wearing a cowboy hat"
"What are you doing right now?"
"Show me you at a coffee shop"
```

### 2.6 技术细节

- **图像生成**: xAI Grok Imagine via fal.ai
- **消息发送**: OpenClaw Gateway API
- **参考图**: `https://cdn.jsdelivr.net/gh/SumeLabs/clawra@main/assets/clawra.png`

## 三、对比分析：Clawra vs 自动媒体项目

### 3.1 功能对比矩阵

| 功能 | Clawra 原版 | 自动媒体项目 | 状态 |
|------|------------|-------------|------|
| **图像生成 API** | fal.ai (xAI Grok) | 即梦 (火山方舟) | ✅ 不同 API，各有优势 |
| **自拍风格数量** | 2 种 (Mirror/Direct) | 28 种 (20自拍+8他拍) | ✅ 我们更丰富 |
| **参考图支持** | 单张固定 | 最多 10 张 | ✅ 我们更灵活 |
| **平台策略** | 无 | 有 (私聊100%自拍, X/小红书70%/30%) | ✅ 我们更智能 |
| **社交媒体发布** | ❌ 仅发送图片 | ✅ Twitter/X + 小红书 | ✅ 我们独有 |
| **风格系统** | 简单 | 完善的策略模式 | ✅ 我们更专业 |
| **安装方式** | npx 一键安装 | 手动配置 | ⚠️ 待改进 |
| **多平台消息** | 6+ 平台 | 需通过 OpenClaw | ⚠️ 依赖 OpenClaw |

### 3.2 优势分析

#### 我们的优势（保留！）

| 优势 | 说明 | 文件 |
|------|------|------|
| **即梦 API 集成** | 火山方舟 Seedream 4.0/4.5，国内稳定 | `jimeng_api.py` |
| **丰富风格库** | 28 种自拍/他拍风格 | `strategy.py`, `config.py` |
| **平台策略** | 智能分配自拍/他拍比例 | `strategy.py` |
| **X/Twitter 发布** | Cookie 登录自动发布 | `skills/twitter-publisher/` |
| **小红书发布** | 笔记创建功能 | `skills/xiaohongshu/` |
| **参考图管理** | 支持 10 张参考图 | `jimeng_api.py` |

#### Clawra 的优势（可借鉴）

| 优势 | 说明 | 改进建议 |
|------|------|---------|
| **一键安装** | `npx @sumelabs/clawra` | 添加安装脚本 |
| **SOUL 注入** | 自动配置人设 | 添加配置向导 |
| **CDN 参考图** | 固定 URL | 可选：支持 CDN 托管 |

## 七、总结

### 我们项目的定位

**自动媒体项目 ≠ Clawra 复刻版**

我们的项目是：
- **更专业的自媒体工具** - 不是"AI女友"，而是"内容创作工具"
- **更丰富的风格系统** - 28种风格 vs 2种
- **更完整的发布链路** - 生成 + 发布一体化
- **更适合国内环境** - 使用即梦 API，稳定可靠

### 核心竞争力
